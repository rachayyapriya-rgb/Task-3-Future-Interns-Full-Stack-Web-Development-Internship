# Lumière Salon – Premium Beauty Studio Website

### Task 3 – Future Interns Full Stack Web Development Internship

## About

Lumière Salon is a premium beauty studio website developed as part of the Future Interns Full Stack Web Development Internship. The website is designed to provide an elegant online presence for a beauty salon, showcasing services, team members, gallery, and contact information through a modern and responsive interface.
